# ProCure AI | Defense Intelligence Portal

ProCure AI is a high-authority, data-dense government procurement platform designed for secure tender evaluation, risk intelligence, and cryptographic auditing.

## Key Features
- **Strategic Sourcing & Evaluation**: Deterministic AI-powered vetting of tender compliance.
- **Risk Intelligence**: Advanced network analysis for cartel and collusion detection.
- **Compliance Ledger**: Cryptographically secured immutable audit trails with SHA-256 hashing.
- **Corrigendum Tracking**: Visual version control and diff viewing for tender revisions.

## Technology Stack
- **Frontend**: Next.js 15, React, Tailwind CSS, Lucide React, Shadcn/UI
- **Backend**: FastAPI (Python), Gemini AI (LLM Integration)
- **Styling**: Midnight Navy & Slate Gray Enterprise Aesthetic

## Getting Started

### Prerequisites
- Node.js 20+
- Python 3.10+

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/bbsarada07/ProCure.git
   ```
2. Install frontend dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## Security & Compliance
All operational vectors are cryptographically anchored to ensure non-repudiation and transparency for supreme audit authorities.
